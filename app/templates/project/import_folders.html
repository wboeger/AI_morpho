Step 1: Extract Landmarks with ImageJ

Download an ImageJ macro pre-configured for this project. Open it in ImageJ/Fiji via Plugins → Macros → Run... The macro guides you interactively through image enhancement, contour extraction (wand tool), and placement of 3 anatomical landmarks per specimen. It outputs CSV files (100 equidistant pseudolandmarks) that you import in Step 2 below.

Marginal Hook Macro

Landmarks: L1 = Point tip, L2 = Toe tip, L3 = Junction Point-Shaft (inner face)

Download Hook Macro (.ijm)

Anchor Macro

Landmarks: L1 = Point, L2 = External tip superficial root, L3 = Distal-most base deep root

Download Anchor Macro (.ijm)
Macro workflow details
  1. Session setup: configure input/output directories, enhancement settings, landmark count (100)
  2. Image review: accept, reject (with reason), or skip each image
  3. Crop & enhance: draw bounding rectangle, optional Gaussian blur + CLAHE + Unsharp Mask, 3× upscale
  4. Orientation: verify structure points right (optional horizontal flip)
  5. B&W conversion: optional threshold-based binary mask for cleaner contour extraction
  6. Wand tool: click on outline to extract contour; adjustable tolerance
  7. Contour smoothing: 3-point moving average (configurable passes)
  8. Landmark placement: click to place L1, L2, L3 sequentially
  9. Equidistant resampling: 100 points starting from L1, spaced by arc length
  10. Verification & editing: review color-coded overlay, optionally edit individual points
  11. Output: CSV files saved (X,Y per row); QC and rejection logs

Full backward navigation is available at every stage.
